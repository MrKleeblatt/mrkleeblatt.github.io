Allgemein

FourFoil ist eine Programmiersprache mehrerer Programmierparadigmen, die nach dem Motto "einfach, aber schnell" erstellt wurde und kontinuierlich weiterentwickelt wird. Sie soll so viele Funktionalitäten wie Möglich enthalten, unter anderem Prinzipien aus der funktionalen, wie auch aus der prozeduralen Programmierung enthalten sein. Eine besondere Rolle spielt in FouFoil auch das Definieren eines Datentypen.

Jedes Programm startet in einer main.ff-Datei, oder einer main(var args string){}-Funktion. Programme bestehen aus Tokens. Tokens der Sprache FourFoil sind:

Für jede Klammer-auf muss es in der Datei eine Klammer-zu geben. Andere Tokens haben andere Syntaxregeln, die beachtet werden müssen, die im Laufe dieser Erklärung noch angesprochen werden. Mehrere Tokens in einer bestimmten Syntax zusammen formieren Anweisungen. Kommentare jedoch werden zwar mit einer Abfolge an Tokens eingeleitet, gehören allerdings nicht zu Anweisungen, da sie vom Compiler komplett ignoriert werden, und nur zum Verständnis des Programmierers beitragen. Eingeleitet wird ein Kommentar mit einem Rautenzeichen und einem darauffolgenden Leerzeichen. Alles, was in dieser Zeile folgt, gilt nun als Kommentar und wird vom Compiler ignoriert. 

            # Das ist ein Kommentar über eine gesamte Zeile xD

Nach jeder Anweisung muss entweder eine neue Zeile beginnen, oder ein Semikolon stehen. Anweisungen können entweder das Ausführen von Funktionen oder von Operator-Anweisungen sein. Operatoren sind Tokens, die eine bestimmte Wirkung vollbringen, also dazu fähig sind, die Daten, die mithilfe anderer Tokens gegeben werden, zu manipulieren. Literale sind eine spezielle Form der Tokens, welche Informationen enthalten. Zahlen sind grundsätzlich immer Literale in FourFoil, wie auch strings, also Zeichen innerhalb zweier Anführungszeichen. Andere Token sind z.B. für die Definition von Variablen zuständig und werden Schlüsselwörter genannt.

Variablen

Variablen sind veränderliche Werte, also reservierte Speicherbereiche, deren Inhalt sich im Lauf des Programms ändern kann. Sie werden mit dem Schlüsselwort "var" erstellt, worauf der Name der Variable folgt. Namen müssen mit einem Unterstrich "_", oder einem der Buchstaben des lateinischen Alphabets beginnen. Nach dem Namen kann entweder direkt ein Wert mittels des Zuweisungsoperatoren "=" zugewiesen werden, oder der (nicht-abstrakte) Datentyp der Variable angegeben werden, worauf auch wieder eine Zuweisung folgen darf. 

            var myVariable = 5
            var my_variable string
            var one int = 1

Um Schreibarbeit zu verkürzen, ist es außerdem möglich, mehrere Variablen des gleichen Types oder Wertes mit einer Operation zu definieren. Hierzu schreibt man wie gewohnt das Schlüsselwort var, worauf jedoch eine normale Klammer-auf folgt, die Namen der zu erstellenden Variablen mit Kommata voneinander getrennt aufgeführt werden, und die Aufzählung mit einer Klammer-zu und entweder einer Deklaration oder Zuweisung abgeschlossen wird. 

            var (number_1,
                number_2,
                number_3) int
            var (one, two) = 3

Pointer

Pointer sind eine spezielle Variante von Variablen, die jedoch keinen Wert selbst enthalten, sondern auf eine andere Variable "zeigen". Den Wert eines Pointers zu verändern, ist gleichzusetzen mit der direkten Änderung der Variable, auf die der Pointer zeigt. Syntaktisch wird ein Pointer mit dem Schlüsselwort "poi" erstellt, worauf ein Name folgt, der den gleichen Voraussetzungen wie ein Variablenname entsprechen muss. Nach dem Namen folgt der (möglicherweise abstrakte) Datentyp der Variable, auf die der Pointer zeigen soll. Sollte der Pointer auf nichts (null) zeigen dürfen, muss das anhand eines Fragezeichens nach dem Datentypen symbolisiert werden. Danach kann dem Pointer eine Adresse zugewiesen werden, wobei es sich um eine Hexadezimalzahl handelt. Um die Adresse einer Variable zu bekommen, gibt man den Variablennamen mit dem Adressenoperatoren "$" davor an.

Anstatt der Angabe des Datentypen kann auch direkt eine Zuweisung erfolgen, wobei allerdings auf die Möglichkeit des null-pointers verzichtet werden muss. Auch bei Pointern können wie bei Variablen mehrere gleichen Types gleichzeitig erstellt werden. 

            var myVariable = 15
            poi my_pointer = $myVariable
            poi my_pointer_2 int?
            poi (pon, pot) string? = null

Operatoren

Operatoren folgen keinem speziellen Muster, sondern werden in der folgenden Tabelle aufgeführt:

Operator-Anweisungen

Operator-Anweisungen sind Operatoren mit anschließender Zuweisung, und somit eine Kurzschreibweise der Kombination zweier Operatoren.

Operator-Anweisung Funktion
a += b Zu der Variable a wird b addiert
a -= b Von a wird b subtrahiert
a *= b a wird mit b multipliziert
a /= b a wird durch b geteilt
a %= b Der Rest, der bei einer Ganzzahldivision von a mit b rauskommt, wird in a gespeichert
a **= b a wird b-mal mit sich selbst multipliziert
++a Kurzform für a += 1
--a Kurzform für a -= 1
a++ Gibt einen neuen Wert zurück, der dem alten Wert von a entspricht, und erhöht a um 1
a-- Gibt einen neuen Wert zurück, der dem alten Wert von a entspricht, und senkt a um 1

Neben den numerischen Anweisungen gibt es auch noch bitweise Operator-Anweisungen, welche das Verhalten der einzelnen Bits einer Zahl ändern, und hier aufgeführt sind:

Operator-Anweisung Funktion
a &= b Führt für jedes Bit der beiden Parameter a und b die UND-Überprüfung durch, und speichert das Ergebnis in a.
a |= b Führt für jedes Bit der beiden Parameter a und b die ODER-Überprüfung durch, und speichert das Ergebnis in a.
a ^= b Führt für jedes Bit der beiden Parameter a und b die EXKLUSIV-ODER-Überprüfung durch, und speichert das Ergebnis in a.
a <<= b verschiebt die Bits von a um b Stellen nach links (entspricht einer Multiplikation mit 2**b), ist aber wesentlich effizienter.
a >>= b Verschiebt die Bits von a um b Stellen nach rechts (entspricht einer Division mit 2**b), ist aber wesentlich effizienter.

Arrays

Arrays sind Datenstrukturen, die mehrere Werte des gleichen Typen enthalten. Man kann sie als einfache Listen ansehen, oder als Vektoren der Mathematik, da mit ihnen genau die gleichen arithmetischen Operationen durchgeführt werden können wie mit mathematischen Vektoren.

Erstellt werden Arrays über das Schlüsselwort arr, auf welches dann der Name des Arrays folgt, dann der (möglicherweise abstrakte) Datentyp und letztendlich die Anzahl an Elementen, die maximal drin gespeichert werden können soll, also quasi die Größe des Arrays. Falls diese anfangs nicht bekannt sein sollte, kann stattdessen auf eine komplexere Datenstruktur zurückgegriffen werden, bei welcher die Größe flexibel ist, die dafür aber natürlich auch mehr Speicher in Anspruch nehmen. 

            arr mein_array string 6
            arr dein_array pointer 42

Anonyme Arrays sind Arrays ohne Namen, die ausschließlich als Parameter übergeben werden können. sie werden durch die folgende Syntax erstellt. Da sie keinen Namen besitzen, ist es dem Programmierer nicht möglich, sie in dem Aktionsradius (z.B. der Funktion) nach der Erstellung anzusprechen. 

            [string "", ""]

Anonyme Arrays können allerdings auch genutzt werden, um ein Array direkt zu initialisieren. Dazu schreibt man das Schlüsselwort arr, den Namen des Arrays, das Gleichheitszeichen und daraufhin ein anonymes Array, welches zugewiesen werden soll. Die Syntax ist wie folgt. 

            arr dein_array = [int 1, 5]

Mit Arrays kann auch gerechnet werden, jedoch sind die Rechenmethoden eingeschränkt. Der Betrag eines Arrays ist die Wurzel aus den Quadraten der einzelnen Elemente, vorausgesetzt, die Elemente sind alle vom Typen number.

Bei der Addition/Subtraktion zweier Arrays werden die Werte nach ihrem Index addiert/subtrahiert. Sollten die Arrays unterschiedlich groß sein, wird in der Rechnung vom größten Array ausgegangen, und das kleinere Array durch eins der passenden Größe ersetzt, welches dann mit Nullen am Ende aufgefüllt wird.

Addition

                [int 1, 2] + [int 3, 3, -1]
                # Ergebnis: [int 4, 5, -1]

Subtraktion

                [int 1, 2] - [int 3, 3, -1]
                # Ergebnis: [int -2, -1, 1]

Bei skalarer Multiplikation werden alle Elemente des Arrays mit dem skalaren Multiplikator multipliziert. Da sich hier auch nicht die Länge des Arrays nicht ändern kann, kann man auch hier eine direkte Zuweisung machen.

Skalare Multiplikation

                [int 1, 2] * 3
                # Ergebnis: [int 3, 6]
                arr mein_array = [int 1, 2]
                mein_array *= 3

Skalarprodukt

                [int 1, 2] * [int 3, 3, -1]
                # Ergebnis: 9

Vektorprodukt

                [int 1, 2] ** [int 3, 3, -1]
                # Ergebnis: ...
